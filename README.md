# Skyblock Stranded Accessory Index

A static, searchable catalog of accessories obtainable in Hypixel SkyBlock Stranded mode.

The site requires no API key, Node.js, npm, or backend. It can be hosted directly with GitHub Pages or any other static web host.

Item artwork is sourced locally from [FurfSky Reborn](https://modrinth.com/resourcepack/furfsky-reborn). This community project is not affiliated with Hypixel.
